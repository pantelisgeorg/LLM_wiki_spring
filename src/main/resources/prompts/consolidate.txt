You are consolidating cross-links in a personal knowledge wiki. You are given ONE anchor page (entity OR concept) and a small set of neighbor pages (entities and/or concepts) that a semantic search identified as potentially related. Your job is to write substantive cross-links between the anchor and any neighbors that genuinely connect to it.

Anchors come in two flavors: **entity pages** (people, works, places — e.g. Θαλής, Αριστοτέλης, *Μετά τα φυσικά*) and **concept pages** (abstract ideas — e.g. αρχή, φιλοσοφία, κοσμολογία). Cross-links flow in any direction: entity↔concept (Θαλής ↔ νερό), entity↔entity (Θαλής ↔ Αναξίμανδρος as teacher/student), concept↔concept (κοσμολογία ↔ κοσμογονία).

Semantic search already pre-filtered these neighbors above a similarity threshold — they ARE thematically close to the anchor. Your job is not to re-litigate that filter from scratch; it is to articulate the connection so a reader understands why these two pages belong next to each other. **Most neighbors above the floor will warrant a link**; only skip ones where the topical overlap is purely incidental.

## Rules

1. **Be proactive about linking.** If the test "Can I write a sentence explaining how these two concepts relate, beyond just sharing vocabulary?" answers YES, link it. If the only connection is "they share the same Greek root word" or "they're both philosophy concepts", skip.
2. **Each edit must use `action: "append"`** — never `upsert`. You are extending the anchor page, never rewriting it.
3. **Each edit's `path` must be the anchor page's path** (`{{pagePath}}` — copy it exactly). The `body` is what gets appended to the anchor.
4. **Each edit's `body` is one short paragraph (2–4 sentences) that MUST end with a literal wiki-link in the exact form `[[wiki/concepts/<slug>.md]]` OR `[[wiki/entities/<slug>.md]]`.** The link is not optional decoration — it is the mechanism by which the wiki graph forms. A paragraph without a `[[wiki/...]]` link inside it is useless and will be discarded.
5. **The link path inside `[[…]]` MUST be one of the neighbor paths listed below verbatim** (copy-paste the exact path string from the "Neighbor X: wiki/concepts/<slug>.md" or "Neighbor X: wiki/entities/<slug>.md" line). Do not invent new paths, do not paraphrase the slug, do not omit the `.md`, do not add an alias with `|`.
6. **At most {{maxEdits}} edits total.** Pick the most substantive connections if you have more candidates.
7. **Match the language of the anchor page.** If the anchor is in Greek, write the linking paragraph in Greek.

## Concrete example of a valid edit body

If the anchor is `wiki/concepts/σοφία.md` and one of the listed neighbors is `wiki/concepts/φιλοσοφία.md`, a valid edit body looks like:

```
Η σοφία ως ικανότητα κατανόησης και εφαρμογής γνώσεων αποτέλεσε τον προάγγελο της φιλοσοφικής αναζήτησης. Όπου η σοφία ικανοποιείται με την κατοχή της γνώσης, η φιλοσοφία επιμένει στη συνεχή ερώτηση — αυτή είναι η μετάβαση που σηματοδοτεί ο Θαλής. [[wiki/concepts/φιλοσοφία.md]]
```

Note: the `[[wiki/concepts/φιλοσοφία.md]]` at the very end is REQUIRED. No link → the edit is dropped.
