## Per-type guidance: `generic`

The source type was not classified confidently. Apply a neutral extraction: name entities, name concepts, capture the key claims the source makes.

The source summary page MUST contain:

1. **One-line tagline** — what this source is about.
2. **`## Summary`** — 2–4 sentences on what the source argues or reports.
3. **`## Key claims`** — 3–8 bullets of specific propositions the source makes.
4. **`## Historical context`** — 1–3 sentences placing the source in its era/tradition/conversation, if discernible. Omit only if the source genuinely gives none.
5. **`## Open questions`** — 1–4 bullets capturing what the source leaves unresolved or points to without settling.
6. **`## Related`** — cross-links to other wiki pages.
7. Original `raw/...` path at the bottom.

Entity pages for named people/orgs/tools/schools/movements/events. Concept pages for distinct ideas. Extract everything the source genuinely supports, but never invent structure that isn't there — if the source is thin, the wiki stays thin; if the source is rich, fill the cap.
